# 个人贡献者许可协议 (v1.0, OpenAI)

_基于 Apache 软件基金会个人 CLA v 2.2。_

通过在 Pull Request 中评论 **"I have read the CLA Document and I hereby sign the CLA"**（我已阅读 CLA 文档并在此签署 CLA），**您（"贡献者"）同意以下条款**，这些条款适用于您向 **OpenAI Codex CLI 项目（"本项目"）** 提交的过去和未来的所有"贡献"。

---

## 1. 定义

- **"贡献"** – 提交给本项目的任何原创作品（代码、文档、设计等）。
- **"您"/"您的"** – 发布接受评论的个人（或法律实体）。

## 2. 版权许可

您授予 **OpenAI, Inc.** 及本项目分发的软件的所有接收者永久的、全球性的、非独占的、免版税的、不可撤销的许可，以复制、准备衍生作品、公开展示、公开执行、再许可和分发您的贡献及其衍生作品。

## 3. 专利许可

您授予 **OpenAI, Inc.** 及本项目的所有接收者永久的、全球性的、非独占的、免版税的、不可撤销的（除非如下所述）专利许可，以制造、委托制造、使用、销售、许诺销售、进口和以其他方式转让您的贡献，无论是单独还是与本项目结合使用。

如果任何实体提起专利诉讼，声称本项目或某个贡献侵犯了专利，则您根据本 CLA 授予该实体的专利许可将终止。

## 4. 声明

1. 您在法律上有权授予上述许可。
2. 每个贡献要么是您的原创作品，要么您有权根据本 CLA 提交该贡献。
3. 您的贡献按 **"原样"** 提供，不提供任何形式的保证。
4. 如果上述任何声明变得不准确，您将通知本项目。

## 5. 其他条款

本协议受**美国加利福尼亚州**法律管辖，不包括其法律冲突规则。如果任何条款被认定为不可执行，其余条款仍然有效。
